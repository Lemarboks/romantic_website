# Cute Love Website

This is a React, TypeScript, and CSS website made for your girlfriend.

## Open It

Double-click `start-website.bat`, then open:

```text
http://127.0.0.1:5174
```

Do not open `index.html` directly from the folder. This project uses Vite, so it needs the local dev server to display correctly.

## Commands

```bash
npm.cmd install
npm.cmd run dev -- --port 5174
npm.cmd run build
```

The photos are in `public/photos`.

## GitHub Pages

For the easiest hosting setup, use **Settings > Pages > Deploy from a branch**, then choose:

```text
Branch: main
Folder: /docs
```

The hosted URL will be:

```text
https://lemarboks.github.io/romantic_website/
```
